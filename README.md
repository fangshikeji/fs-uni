# Fs-uni

fs-uni是一款由方是科技前端部研发的一款高质量的uniapp ui组件库
文档地址：http://unidoc.sxidc.com/

# 快速开始

无需import,直接使用即可

eg：<fs-avatar></fs-avatar>

# 使用cli

1、安装

 npm install -g fskj-cli

2、初始化

fs-cli init <项目名>
